# Japan Bathing Journal

Editorial site: sauna, onsen and sento for the Western traveler, written by a Japanese sauna producer.

Published via GitHub Pages. Static HTML/CSS, no build step, no external API dependencies.

Source of truth for content is `vault/output/japan-bathing-journal/` in the private working repo — this repo is a public-facing mirror of that folder's `index.html` and `articles/`, kept in sync manually before each deploy.
